# 玩具工廠 / Toy Forgotten

> 一間收容被遺忘玩具的工廠，一個試著找回記憶的布偶。

《玩具工廠》是一款以**探索、修復與敘事**為核心的 2D 劇情遊戲。
玩家白天在廢棄工廠中翻找角落，尋找受損的玩具與記憶碎片；
晚上在工作台修復它們，聽見每件玩具背後的故事，
同時也拼回主角「結依 / Yui」失落的過去。

**開發狀態：** 早期原型 · Godot 4 · GDScript · 目標平台 Steam（PC）

---

## 世界觀

玩具世界與人類世界平行存在。玩具有情感，有些甚至承載了人類殘留的靈魂。  
這座工廠表面上是收容被退貨、被遺忘、被損壞玩具的地方——  
但它同時也是一個陷阱，藏著主角不願面對的真相。

---

## 核心玩法

```
白天 → 探索工廠
         翻找箱子、角落、隱藏空間
         發現破損玩具、零件、記憶碎片

晚上 → 修復工作台
         用找到的零件修復玩具
         聽見玩具的故事，觸發主角的記憶
         決定玩具的去留，解鎖明天的新區域
```

---

## 目前可玩內容（Milestone 1 完成）

1. 開場過場動畫
2. 進入廢棄工廠場景，自由移動探索
3. 靠近時鐘人，按 `E` 開始互動
4. 閱讀修復前對話（打字機效果）
5. 進入時鐘指針修復小遊戲（拖曳至正確角度）
6. 觸發修復後對話與感謝場景
7. 時鐘人走向角落，播放日記動畫
8. 一天結束過場

---

## 操作方式

| 按鍵 | 功能 |
|------|------|
| `WASD` | 移動 |
| `E` | 互動 / 推進對話 |
| 滑鼠左鍵 | 拖曳修復（小遊戲） |
| `Esc` | 跳過開場過場 |

---

## 執行方式

1. 用 **Godot 4** 開啟專案資料夾
2. 直接執行（主場景已在 `project.godot` 設定）

**專案設定：** 視窗 `1280×720`，Rendering 2D / GL Compatibility
